# MyProject

A standard Zig project demonstrating idiomatic layout.

## Building

```bash
zig build
```

## Running

```bash
zig build run
```

## Testing

```bash
zig build test
```

## Project Structure

```
myproject/
├── build.zig           # Build configuration
├── build.zig.zon       # Package manifest
├── README.md           # This file
├── LICENSE             # License file
└── src/
    ├── root.zig        # Library module root
    └── main.zig        # Executable entry point
```
