# Example 3: Common Pattern Reference

Quick reference for frequently-used Zig patterns.

## Patterns Included

### Initialization
- Struct literals
- Init/deinit pairs

### Error Handling
- try (propagation)
- catch (handling)
- errdefer (cleanup)

### Memory Management
- defer cleanup
- arena for temporary allocations

### Optional Handling
- orelse (default)
- if unwrapping
- while unwrapping

### Iteration
- for with index
- while with continue

## Running

```bash
zig run src/main.zig
```
